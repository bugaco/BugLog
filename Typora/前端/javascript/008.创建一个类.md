# 008.创建一个类，以及继承

## 1.创建一个类

```js
function Person(name) {
  this.name = name;
  this.greeting = function() {
    alert('Hi! I\'m ' + this.name + '.');
  };
}
```

实例化：

```js
var person1 = new Person('Bob');
```

创建一个对象的副本：

```js
var person2 = Object.create(person1);
```



## 2.继承

### (1)有参的继承

```js
function Teacher(name, subject) {
  Person(name);
  this.subject = subject
}
```

### （2）无参的继承

```js
function Brick() {
  this.width = 10;
  this.height = 20;
}
function BlueGlassBrick() {
  Brick.call(this);

  this.opacity = 0.5;
  this.color = 'blue';
}
```
